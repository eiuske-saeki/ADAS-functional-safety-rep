# ADAS Functional Safety Simulation Tool

## 概要
このプロジェクトは、ADAS（先進運転支援システム）の機能安全性をシミュレートし、ASIL（自動車安全度水準）を算出するためのツールです。

## 機能
- シミュレーションデータの生成
- 車両運動のシミュレーション
- ASIL値の計算
- 結果の可視化
